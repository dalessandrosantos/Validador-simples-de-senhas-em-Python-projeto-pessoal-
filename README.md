# Validador simples de senhas em Python

Desenvolvi um validador de senhas em Python com foco em praticar conceitos básicos da linguagem, como estruturas de repetição, condicionais e tratamento de entrada do usuário.

## Funcionalidades

- Verifica se a senha possui pelo menos 6 caracteres (desconsiderando espaços)
- Conta quantas senhas válidas e inválidas foram digitadas
- Valida a opção de continuar (`S`/`N`), tratando entradas incorretas
- Exibe um resumo final com o total de senhas válidas e inválidas

## Como executar

1. Certifique-se de ter o **Python 3** instalado.
2. Clone este repositório ou baixe o arquivo `validador_senha.py`.
3. No terminal, execute:

```bash
python validador_senha.py
```

## Observação

Este projeto é apenas para fins de estudo. Não deve ser usado para armazenar senhas reais em produção.
